# Proyecto_Asistencia_S-S v2
Este script fue desarrollado en VS Code con formato “.py"(python), siendo una versión mejorada con librerias del proyecto con el mismo nombre.
Es un Registro de asistencia simple el cual llamé “Registrador S-S v2”, el cual sirve para ingresar nombre, apellido y sexo junto con su asistencia (presente, atrasado, ausente).
Este programa permite:
1)	Ingresar nombre, apellido, sexo, asistencia: Solicita nombre, apellido y sexo, luego su asistencia.
2)	Mostrar asistencia: Muestra todos los usuarios que han sido ingresados previamente al sistema.
3)  Editar asistencia: Modifica cualquier parámetro ingresado al sistema
4)	Eliminar asistencia: Borra 1 registro a elección del usuario.
5)	Salir: Cierra el programa
A esto también se le suma un CSV para tener un respaldo y tener la lista de forma ordenada
